### 快速安装
1. git clone https://github.com/anyvicp2/AnyVICP2
2. 你需要 NGINX \ APACHE PHP MYSQL
3. 把 clone 下来的东西复制到网站目录
4. 打开网站, 访问 网站/core/install/install.php 来安装
5. 显示 '安装成功' 则可以转回到首页来使用了.
